val puconst : Ast.constant Angstrom.t

puconst parses any (unsigned) constant and returns it as a constant type. Should be used for expression parsing. Cannot parse surrouning whitespaces.

val psconst : Ast.constant Angstrom.t

psconst parses any (probably signed) constant and returns it as a constant type. Should be used for pattern parsing. Cannot parse surrouning whitespaces.
